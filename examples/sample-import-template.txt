# Package Import Template (TXT Format)
# You can use any of these formats:

## Format 1: Key-Value Pairs (Recommended)
Package 1:
Type: Box
Dimensions: 30 x 20 x 15 cm
Weight: 2.5 kg
Fragile: Yes
Priority: High
Destination: 123 Main Street, New York, NY 10001, USA
Sender: John Doe (john@example.com)
Instructions: Handle with care

---

## Format 2: CSV-like (Comma Separated)
Box, 30, 20, 15, cm, 2.5, kg, Yes, High, 456 Oak Avenue, Los Angeles, CA, 90001, USA

---

## Format 3: JSON-like
{
  "type": "Envelope",
  "dimensions": { "length": 25, "width": 18, "height": 1 },
  "weight": 0.2,
  "destination": { "city": "San Francisco", "country": "USA" }
}

---

## Format 4: Natural Text
Send a large box measuring 40x30x25 centimeters weighing 5kg to Seattle, Washington, USA.
Mark it as fragile and high priority.

---

## Format 5: List Format
- Type: Pallet
- Size: 120 x 100 x 150 cm
- Weight: 200 kg
- City: Chicago
- Country: USA
- Priority: Standard

---

# Notes:
# - At minimum, provide city OR country for destination
# - The system will extract available information
# - Missing fields can be filled in later
# - Supports multiple packages (separate with ---)
